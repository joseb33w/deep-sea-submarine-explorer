# deep-sea-submarine-explorer

A browser-based 3D deep sea submarine game with:

- email/password login via Supabase
- real-time 3D submarine piloting with touch look controls
- glowing creatures, wrecks, caves, and thermal vents
- oxygen and battery drain with surfacing/refuel loop
- specimen collection and photo logging
- 3D museum curation and tours
- museum ratings and collection leaderboard

## Files

- `index.html` – app entry
- `styles.css` – UI styling
- `app.js` – main app, 3D scenes, gameplay, UI
- `auth.js` – auth/profile logic
- `api.js` – Supabase data access
- `data.js` – species and POI definitions
- `supabaseClient.js` – Supabase setup

Open `index.html` in preview to play.
